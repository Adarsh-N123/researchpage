    About

    This is the website of the research for computer science at iit goa.



    Computer Science

    Currently, we are working in the following areas of Computer Science and Engineering:
    1. Theoretical Computer Science
    Algorithms, computational complexity, arithmetic complexity, Machine learning, topic models, marine data science, learning strategies, Logic in Computer Science, formal methods of computation, automata theory, Semigroup theory, functional Programming and categories.

    2. Computing System
    Internet of Things, System Modeling and Discrete-event simulation, Software verification, compilers, static analysis and debugging. Computing architecture, reconfigurable computing (FPGA, High Level Synthesis), Computing hardware for AI, computer security, Applications of computing in medical devices and marine technology domains.
